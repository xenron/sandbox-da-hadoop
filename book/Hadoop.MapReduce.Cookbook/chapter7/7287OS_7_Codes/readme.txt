Source package for Chapter 7 : Hadoop Cookbook

Pre-requisites
==============
Apache Ant
Apache Hadoop 1.0.x
Apache HBase
Java 1.5 or above.

Steps to build
==============
1. Export HADOOP_HOME environmental variable pointing to the Hadoop home directory.
> export HADOOP_HOME = /u/hadoop-1.0.4

2. Export HBASE_HOME environmental variable pointing to the HBase home directory.
> export HADOOP_HOME = /u/hbase-0.90.6

3. Use ant to build the package
> ant
